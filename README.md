# Kočička žere myšičky

Toto je jednoduchá webová hra pro jednoho hráče.

## Jak hru spustit

1. Otevři soubor `index.html` v prohlížeči.

2. Pokud se ti hra nenačte ze souboru, spusť v terminálu v adresáři projektu:

   ```bash
   python3 -m http.server 8000
   ```

   Pak otevři v prohlížeči:

   ```text
   http://127.0.0.1:8000
   ```

3. Alternativně použij ve VS Code rozšíření **Live Server** a spusť jej z `index.html`.

## Ovládání

- šipky na klávesnici

## Poznámka

Kočička i myšičky se pohybují po mapě a nemohou procházet zdmi.
